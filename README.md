## Set Up

### 1. Clone the repository

https:
```sh
git clone https://github.com/hstor3/contact-form.git
```

SSH:
```sh
git clone git@github.com:hstor3/contact-form.git
```

### 2. Install the packages

```sh
npm install
```

### 3. Run Locally

```sh
npm start
```
